# GROUP-03 Mini Project

## Topic: Gamified Habit Tracker

### To get started

####  1. Please run the provided `schema.sql` inside the `resources` dir
####  2. Create a `.env` file in the root of the project
####  3. Provide values to these `.env` keys
```dotenv
DB_USER=your-db-user
DB_PASSWORD=your-db-password
DB_URL=your-db-url
REDIS_HOST=your-redis-host
REDIS_PORT=your-redis-port
JWT_SECRET=your-jwt-32bit-jwt-secret
MAIL_HOST=smtp.example.com
MAIL_PORT=your-mail-port
MAIL_USERNAME=your-mail-username
MAIL_APP_PASSWORD=your-mail-app-password
```
